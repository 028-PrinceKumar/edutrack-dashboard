# 🎓 EduTrack Dashboard

EduTrack is a modern education dashboard designed to track student performance, courses, assignments, streaks, and live classes in an interactive way.

## 🚀 Features

- 📊 Student Performance Dashboard
- 🏆 Leaderboard Ranking System
- 🔥 Daily Streak Tracker
- 🎖️ Badges & Achievements
- 📚 Course Progress Monitoring
- 📝 Assignment Tracker (Pending, Submitted, Late)
- 🎥 Live Class Interface with Chat & Hand Raise

## 🛠️ Tech Stack

- React.js (CDN based)
- Tailwind CSS
- JavaScript (ES6)
- HTML5

## 📸 Screens Included

- Dashboard Overview
- Leaderboard
- Course Progress
- Assignments Tracker
- Live Class UI

## 💡 How It Works

- Tracks student progress across multiple subjects
- Calculates course completion percentage
- Displays rankings based on points & activity
- Shows streak consistency and achievements
